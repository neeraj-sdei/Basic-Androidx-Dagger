# Basic-Androidx-Dagger

This provides a basic app structure for kickstarting any new project. 
App structure comes with:
- predefined logging for debug and release.
- predefined retrofit for network calls
- some basic modules like app and network modules added using Dagger 

## Libraries used :

- For Logging [Timber](https://github.com/JakeWharton/timber)
- For network calls [Retrofit](https://square.github.io/retrofit/)
- For dependency injection [Dagger 2](https://google.github.io/dagger/android)
